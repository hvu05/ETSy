## ETSy (Event Ticket System)
Đây là hệ thống hỗ trợ bán vé chịu tải lớn (< 100.000 lượt truy cập tại thời điểm mở bán vé). Hệ thống được build bằng kiến trúc Microservice, gồm các service chính: Auth service, Booking service, Search Service, Payment service. Chi tiết hơn được đặc tả trong các file markdown.
